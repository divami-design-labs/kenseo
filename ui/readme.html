Kenseo Development Team Notes

Note:

We are using Backbone.js to write the code in MV* structure. So, please follow backbone structure and use lodash to full extent wherever utils functions are needed.

Getting Started:

  • Keep the project folder in C:\xampp\htdocs\ with name Kenseo.
  • Install Node.js
  • CD to project folder in CLI and enter npm install.
  • Now you are all set up.
  • Now run gulp watch to non-stop watch for file changes in sass and html templates.

Templating:

For templating, we are using gulp to watch the template changes. Please check /ui/gulpfile.js to understand how it works.

Adding a Template:

  • Add .html file of the template in app/templates folder with a recognizable name say "my-first-template.html"
  • To use the templates, use sb.setTemplate method with arguments templateName and data to replace in the template.

Standardizing Timezones:

To maintain standard time throughout frontend and backend:

  • we are storing the time in UTC format in php.

  • In mysql, it is in local timezone but while fetching we need to convert it to UTC format as shown below:

    SELECT 
    CONVERT_TZ(
        <date_column>, 
        @@session.time_zone, 
        '+00:00'
    ) 
FROM <table_name>

  • In Frontend, always add the local timezone to the fetched time from backend. 

    sb.addTimeZoneToDate(date)
