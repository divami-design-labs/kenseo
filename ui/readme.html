Kenseo Development Team

Note:

We are using Backbone.js to write the code in MV* structure. So, please follow backbone structure.

Getting Started:

  • Keep the project folder in C:\xampp\htdocs\ with name Kenseo.
  • Install Node.js
  • Install Ruby 1.9.3p551 (stable version)
  • Install sass through Ruby command prompt. gem install sass.
  • Double click Kenseo/ui/builds/Installer.bat file.
  • Wait till all the files are installed.
  • Now you are all set up.
  • Now, whenever working on project run /ui/builds/Runner.bat file. Which will automatically run apache and watch for template changes and sass.

Templating:

For templating, we are using gulp to watch the template changes. Please check /ui/gulpfile.js to understand how it is working.

Adding a Template:

  • Add .html file of the template in app/templates folder with a recognizable name say "my-first-template.html"
  • To use the templates, use sb.renderTemplate or sb.renderXTemplate.
